# drawingprogram

drawingprogram (very original name) is a tiny drawing program inspired by Drawception's sandbox. It's made with only HTML5 and JS, with no additional libraries. I made it in like a couple days.

Currently the redo button isn't implemented (but the undo is.). I also want to add other features, such as customizable palettes and maybe even some crazy stuff like a collaborative/multiplayer mode (Probably not, though. Networking is hard)
